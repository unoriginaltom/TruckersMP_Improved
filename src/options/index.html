UI Settings

  • Image Previews Only in Appeals/Reports/Feedbacks commentaries
  • Wider Page Only in Appeals/Reports/Feedbacks, useful for image previews
  • Auto-insert separator In accept-report modal and manual ban page
  • Auto-insert report comments When accepting reports, insert either a localised or own (see "Other settings") accept message
  • Link notifications Small "pop-up" when a link is copied. It does not affect the error alert
  • "View appeal" in new tab Open appeals in new tab for Reports Table
  • "View report" in new tab Open reports in new tab for Reports Table and Other Open Reports forms
  • Ban length Enable ban length feature
  • Enable feedback improvement You can simply disable the feedback improvement. Table and tickets will be not improved after disabling
  • "View feedback" in new tab Open feedbacks in new tab for Feedbacks Table (works only if the feedbacks improvement is activated)
